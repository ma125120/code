  1. 使用vue+ES6+vue-lazyload插件+webpack打包工具+sass预处理器等技术。

react画廊

  1. 使用react+CSS3+ES6制作而成的画廊。
  2. 使用ES6的class进行react组件的编写。
  3. 使用transform-style和perspective使得css动画在3d空间内呈现。
  4. 使用backface-visibility使得图片可翻转。
  5. 通过获取屏幕的大小，来计算图片分布的位置。
  6. 点击中心图片可进行翻转，点击周围的图片可使其居中，并且在这个过程中会伴随有动画，下方的列表项也会随着图片而变化，点击列表项亦可。
  7. 前往浏览

vue网上商城

  1. 使用vue+vuex+vue-router+vue-resource+ES6+webpack等技术。
  2. 通过json文件来获取有关商品的信息，将其存储于vuex。
  3. 点击不同的商品可进入不同的商品详情页。
  4. 进入商品详情页后，点击不同的缩略图可显示相应的大图。在选择好商品后，点击加入购物车即可在购物车进行浏览。
  5. 购物车页面会显示商品名称，颜色，尺码，单价，数量，总价等信息。可以对数量进行修改，全选或取消全选，对这些信息进行的操作都会导致总价变化。
  6. 点击首页可进入商品展示页，点击购物车可进入购物车页面。
  7. 前往浏览

原生js封装

  1. function.js 是对一些简单函数的封装,包括：
    • 获取查询字符串或哈希值的内容
    • 查询浏览器是否支持某个css属性
    • 获取区间内的随机数
    • 获取滚动条高度
    • 实现类式继承
    • 自定义事件
    • 获取CSS样式
    • 关于事件的兼容性处理
    • 模板字符串
  2. tap.js是对移动端触摸手势的一些封装，主要是tap,swipeLeft,swipeRight,swipeTop,swipeTop。
  3. util.js是模仿jq所写的一个js封装库，可实现jq的大部分功能。
